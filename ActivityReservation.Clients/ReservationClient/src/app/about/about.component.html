关于 OpenReservation

OpenReservation，起初的设计和开发是因为学校活动室预约流程希望从之前繁琐低效的完全线下预约 修改为线上预约+线下盖章审批的方式来预约学校的活动室。

目前使用 ASP.NET Core 开发, 使用 Docker + k8s + nginx 部署，
